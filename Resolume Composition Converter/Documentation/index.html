Installation

To install the application, simply drag the "Resolume Composition Converter.app" to your Applications folder.

Getting Started

  1. Launch the application from your Applications folder
  2. Select your input composition file (.avc)
  3. Choose an output location for the converted file
  4. Set your desired resolution and frame rate
  5. Click "Convert Composition"

For more detailed instructions, please refer to the User Manual.
